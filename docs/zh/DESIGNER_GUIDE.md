# 设计师快速上手指南

> SketchUp Agent Harness 设计助手
> 让室内设计师使用自然语言创建 3D 模型

---

## 快速开始

### 1. 启动系统

在终端（Terminal）中运行以下命令：

```bash
cd /path/to/sketchup-agent-harness
./start_designer.sh
```

或者分步运行：

```bash
# 终端 1：启动 SketchUp 桥接
cd su_bridge
ruby -e "load 'lib/su_bridge.rb'; SuBridge::ServerListener.new.start"

# 终端 2：启动设计师助手
cd mcp_server
python3 -m mcp_server.server
```

看到类似这样的输出就表示启动成功了：
```
[su_bridge] Server started on /tmp/su_bridge.sock
```

## 2. 开始一个新项目

对我说："**开始新项目**"

我会主动询问你几个问题：

1. **"你的房间有多大？"** — 告诉我长和宽，比如"5米 x 4米"
2. **"喜欢什么风格？"** — 从下面选一个，或者让我给你推荐：
   - 奶油风（日式+北欧融合，暖暖的）
   - 工业风（酷酷的混凝土+黑色金属）
   - 北欧风（干净的白+木色）
   - 地中海（白色墙壁+蓝色点缀）
   - 波西米亚（彩色异域风）
   - 现代极简（高级灰+黄铜）

3. **"有什么特别需求吗？"** — 比如"要一个能坐6人的餐桌"、"需要很多收纳空间"等

回答完这些问题后，我会自动帮你：
- 创建项目文件夹
- 画好房间的墙体
- 设定好地板和墙面颜色

## 3. 常用指令

### 添置家具

| 你说的话 | 系统做的 |
|---------|---------|
| "放一个双人沙发" | 找到合适的沙发并摆放在客厅中央 |
| "把餐桌放在北墙边" | 自动对齐到北墙内侧 |
| "书桌靠窗放" | 计算窗户位置，对齐摆放 |

### 改变颜色

| 你说的话 | 系统做的 |
|---------|---------|
| "把墙涂成白色" | 找到所有墙面，应用白色涂料 |
| "地板换成木色" | 地板变成温暖的木纹色 |
| "沙发要灰色的" | 沙发变成你喜欢的灰色 |

### 查看效果

| 你说的话 | 系统做的 |
|---------|---------|
| "帮我拍几张照" | 从多个角度自动拍摄 |
| "来个全景图" | 全景俯瞰整个房间 |
| "看看客厅的视角" | 切换到客厅内部视角 |

### 调整布局

| 你说的话 | 系统做的 |
|---------|---------|
| "把餐桌往左移一点" | 调整位置（会保持和其他物体的间距） |
| "餐桌转个方向" | 旋转到合适的角度 |
| "删掉这把椅子" | 移除选中的物品 |

---

## 4. 撤销操作

如果做错了，说：

- **"撤销"** — 撤销上一步操作
- **"回到上上次"** — 连续撤销
- **"重做"** — 恢复刚才撤销的操作

所有操作都是可以撤销的，所以放心尝试！

## 5. 查看设计方案

完成后，我会把方案保存到 `designs/你的项目名/` 文件夹里：

```
designs/
└── 项目名/
    ├── v1.0_initial/
    │   ├── model.skp          ← SketchUp 文件，可以继续编辑
    │   ├── 全景图.png          ← 俯瞰图
    │   ├── 客厅视角.png        ← 客厅内部视角
    │   └── 物料清单.md         ← 用了哪些家具、材料
    └── v1.1/
        └── ...
```

## 7. 风格参考

### 奶油风（日式+北欧）
暖暖的奶油色墙壁，配合浅木色地板和绿植点缀。温馨舒适，适合小户型。

### 工业风
混凝土质感的墙面，深色金属管道，木质家具。个性十足，适合追求酷感的年轻人。

### 北欧风
纯白色墙壁，浅木色地板，灰色布艺沙发。简约干净，适合喜欢简洁的人。

### 地中海风
白色石灰墙，蓝白色瓷砖点缀，陶土地砖。清爽度假风，适合热带地区。

### 波西米亚风
多彩的织物，异域图案的抱枕，绿植装饰。自由随性，适合追求个性的人。

### 现代极简
高级灰墙面，大理石纹理，黄铜金属点缀。优雅高级，适合追求品质的人。

## 常见问题

**Q: 系统没反应怎么办？**
A: 检查终端是否有错误信息，或者重新启动一次。

**Q: 放错位置了怎么办？**
A: 说"撤销"，然后重新放置。

**Q: 可以换风格吗？**
A: 可以！说"把风格换成工业风"，我会保留家具布局，只改变颜色和材质。

**Q: 想添加自定义家具可以吗？**
A: 可以把 .skp 文件放到指定文件夹，我会自动识别。
