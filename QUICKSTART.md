# 快速入门指南

## 启动系统

### 方式一：一键启动

```bash
cd /path/to/sketchup-claude-code
./start_designer.sh
```

### 方式二：分步启动

```bash
# 终端 1：启动 SketchUp 桥接
cd su_bridge
ruby -e "load 'lib/su_bridge.rb'; SuBridge::ServerListener.new.start"

# 终端 2：启动 MCP 服务器
cd mcp_server
python -m mcp_server.server
```

看到类似输出表示启动成功：
```
[su_bridge] Server started on /tmp/su_bridge.sock
```

## 开始新项目

对我说：**"开始新项目"**

我会问你：

1. **房间尺寸** - "5米 x 4米"
2. **风格选择** - 奶油风/北欧风/工业风/地中海风/波西米亚风/现代极简
3. **特殊需求** - "需要6人餐桌"、"收纳空间要多"

完成后自动帮你创建项目文件夹、绘制墙体、设定颜色。

## 常用指令

### 添置家具

| 你的话 | 系统做的 |
|--------|---------|
| "放一个双人沙发" | 找到合适的沙发并摆放在客厅中央 |
| "把餐桌放在北墙边" | 自动对齐到北墙内侧 |
| "书桌靠窗放" | 计算窗户位置，对齐摆放 |

### 改变颜色

| 你的话 | 系统做的 |
|--------|---------|
| "把墙涂成白色" | 应用白色涂料到所有墙面 |
| "地板换成木色" | 地板变成木纹色 |
| "沙发要灰色的" | 沙发变成灰色 |

### 查看效果

| 你的话 | 系统做的 |
|--------|---------|
| "帮我拍几张照" | 从多个角度自动拍摄 |
| "来个全景图" | 俯瞰整个房间 |
| "看看客厅视角" | 切换到客厅内部视角 |

### 调整布局

| 你的话 | 系统做的 |
|--------|---------|
| "把餐桌往左移" | 调整位置 |
| "餐桌转个方向" | 旋转到合适角度 |
| "删掉这把椅子" | 移除选中的物品 |

## 撤销操作

- **撤销** - 撤销上一步
- **回到上上次** - 连续撤销
- **重做** - 恢复刚才撤销的操作

## 保存和查看

设计完成后，方案保存在 `designs/项目名/`：

```
designs/项目名/
├── v1.0/
│   ├── model.skp          # SketchUp 文件
│   ├── 全景图.png          # 俯瞰图
│   └── 物料清单.md         # 家具和材料清单
```
